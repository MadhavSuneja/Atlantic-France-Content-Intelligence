# Atlantic France Top 50 Content Intelligence

## Project Overview

This project analyzes the Atlantic France Top 50 music playlist dataset using Python, Pandas, Plotly, and Streamlit.

The objective is to identify audience preferences, explicit content trends, album format preferences, duration patterns, and ranking characteristics through exploratory data analysis and an interactive dashboard.

## Features

* Explicit vs Clean Content Analysis
* Content Compliance Summary Panel
* Album Format Distribution Analysis
* Song Duration Analysis
* Rank Tier Comparison (Top 10, Top 25, Top 50)
* Interactive Dashboard Filters
* Download Filtered Data

## Technologies Used

* Python
* Pandas
* Plotly Express
* Streamlit
* Jupyter Notebook

## Files

* App.py – Streamlit dashboard
* Atlantic_France.csv – Raw dataset
* Atlantic_France_Processed.csv – Processed dataset
* France_Top50_Analysis.ipynb – Exploratory Data Analysis notebook

## Dashboard Capabilities

### Filters

* Date Range Selector

### Analytics

* Explicit vs Clean Content Distribution
* Album Type Distribution

## Key Performance Indicators (KPIs)

* Explicit Content Share
* Clean Content Dominance Ratio
